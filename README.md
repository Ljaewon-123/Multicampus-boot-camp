# 크롤링(Crawling)

- Open API활용 해서 크롤링
- 스타벅스 크롤링
- 스타벅스 크롤링과 Django(장고) Ajax로 활용
- 드라이버(driver) 이용,셀레니움 사용
- Naver크롤링
- 인스타 크롤링,및 인스타 img태그 로 사진다운
- 그외 seaborn과 matplot같은 시각화 패키지 사용
- 기본 folium연습 (추후 프로젝트에서 더 많이 활용하게됨)
- wordcloud연습
